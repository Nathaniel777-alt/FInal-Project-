HTML Cheat Sheet

  • <html> – Root element
  • <head> – Metadata container
  • <body> – Page content
  • <h1> to <h6> – Headings
  • <p> – Paragraph
  • <a> – Hyperlink
  • <img> – Image
  • <ul> / <ol> – Lists
  • <li> – List item
  • <div> – Container
  • <span> – Inline container
  • <table> – Table
  • <form> – User input
  • <input> – Form field
  • <button> – Clickable button
